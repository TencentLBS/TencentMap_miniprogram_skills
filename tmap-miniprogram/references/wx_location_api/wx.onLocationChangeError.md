# wx.onLocationChangeError(function listener)

**基础库支持**：2.19.5 开始支持（低版本需做兼容处理）。

## 功能描述

监听持续定位接口返回失败时触发。

## 参数

### function listener

监听持续定位错误的回调函数。

该回调函数接收一个对象参数 `res`，包含以下属性：

| 属性 | 类型 | 说明 |
| :--- | :--- | :--- |
| errCode | number | 错误码 |

## 其他说明

- **小程序插件**：不支持
- **微信鸿蒙 OS 版**：支持

## 注意事项

文档中未包含具体的示例代码以及 `errCode` 对应的错误码详细列表。
